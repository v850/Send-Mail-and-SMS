# Send-Mail-and-SMS

Use api key from sms-magic api. I have executed it with my API KEY

install packages 

to deal with requests to API 

``
pip install requests
``

install csv module to deal with given csv file data

``
pip install csv
``

run command in command prompt so it acts as server to send and recieve Emails.

Assuming mail server runs on localhost on port 1025

``
python -m smtpd -c DebuggingServer -n localhost:1025
``

Now execute the  program
